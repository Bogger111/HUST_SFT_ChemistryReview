# 大学化学常用常数速记表

这份只整理考试、作业和公式计算里最常用的常数。优先背“常用值 + 单位 + 什么时候用”，不要只背数字。

## 一、最优先记忆

| 常数 | 符号 | 常用值 | 常见用途 | 易错提醒 |
|---|---|---:|---|---|
| 摩尔气体常数 | \(R\) | \(8.314\ \mathrm{J\cdot mol^{-1}\cdot K^{-1}}\) | 热力学、电化学、气体、Arrhenius | 与 J、Pa·m\(^3\) 配套 |
| 摩尔气体常数 | \(R\) | \(8.314\ \mathrm{kPa\cdot L\cdot mol^{-1}\cdot K^{-1}}\) | \(pV=nRT\) 中 \(p\) 用 kPa、\(V\) 用 L | \(1\ \mathrm{L}=1\ \mathrm{dm^3}\) |
| 摩尔气体常数 | \(R\) | \(0.08206\ \mathrm{L\cdot atm\cdot mol^{-1}\cdot K^{-1}}\) | 气体题中压力用 atm | 不要和 kPa 混用 |
| Avogadro 常数 | \(N_A\) | \(6.022\times10^{23}\ \mathrm{mol^{-1}}\) | 粒子数与物质的量换算 | \(N=nN_A\) |
| Faraday 常数 | \(F\) | \(9.6485\times10^4\ \mathrm{C\cdot mol^{-1}}\) | \(\Delta G=-nFE\)、电解计算 | 可近似 \(9.65\times10^4\) |
| 元电荷 | \(e\) | \(1.602\times10^{-19}\ \mathrm{C}\) | 电荷、电势能、电子加速 | \(F=N_Ae\) |
| Planck 常数 | \(h\) | \(6.626\times10^{-34}\ \mathrm{J\cdot s}\) | \(E=h\nu\)、\(\lambda=h/p\) | 单位不能漏 \(s\) |
| 光速 | \(c\) | \(2.998\times10^8\ \mathrm{m\cdot s^{-1}}\) | \(\Delta E=hc/\lambda\) | nm 要换算成 m |
| Boltzmann 常数 | \(k_B\) | \(1.381\times10^{-23}\ \mathrm{J\cdot K^{-1}}\) | 微观热运动、统计热力学 | \(R=N_Ak_B\) |

## 七、速记口诀

| 口诀 | 含义 |
|---|---|
| 气体用 \(R\)，看单位先选 \(R\) | J、kPa·L、L·atm 三套不要混 |
| 电化学先找 \(n\)，再用 \(F\) | \(\Delta G=-nFE\)、\(\frac{0.0592}{n}\) 都离不开电子数 |

## 八、最常见单位陷阱

1. \(R=8.314\ \mathrm{J\cdot mol^{-1}\cdot K^{-1}}\) 与 \(pV\) 的 SI 单位配套：\(p\) 用 Pa，\(V\) 用 m\(^3\)。
2. \(R=8.314\ \mathrm{kPa\cdot L\cdot mol^{-1}\cdot K^{-1}}\) 与 \(p\) 用 kPa、\(V\) 用 L 配套。
3. \(R=0.08206\ \mathrm{L\cdot atm\cdot mol^{-1}\cdot K^{-1}}\) 与 atm、L 配套。
4. 用 Nernst 方程的 \(0.0592\) 时，默认温度是 298 K，且对数是 \(\lg\)。
5. \(K_w=10^{-14}\) 和 \(pH+pOH=14\) 默认是 298 K。
6. \(1\ \mathrm{eV}=1.602\times10^{-19}\ \mathrm{J}\)，不要把 eV 当作电压单位。
7. 波长用 \(hc/\lambda\) 时，若 \(hc=1240\ \mathrm{eV\cdot nm}\)，\(\lambda\) 必须用 nm。
8. 标准态热力学常用 \(p^\theta=100\ \mathrm{kPa}\)，不等同于 \(1\ \mathrm{atm}=101.325\ \mathrm{kPa}\)，但很多普通计算中差别可近似忽略。
